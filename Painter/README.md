# Painter 小畫家

使用 C++ 與 SFML 製作的簡易繪圖程式。

## 示範

![demo](demo.gif)

## 技術

- 語言：C++
- 函式庫：SFML 2.x
- IDE：Visual Studio 2022

## 操作方式

- 按住左鍵拖曳：在畫布上繪圖
- 點擊左上角色塊：切換筆刷顏色（白／黑／紅／綠）
- 按 `S` 鍵：將畫布儲存為 `canvas.png`

## 如何執行

1. 安裝 SFML 2.x（建議透過 [vcpkg](https://github.com/microsoft/vcpkg)）
2. 使用 Visual Studio 開啟 `Painter.sln`
3. 建置並執行
